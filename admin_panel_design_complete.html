Layout — fixed sidebar, all screens share this shell
Desktop (≥ 1200px) — primary
Fixed left sidebar 240px. Content area fills remaining width. Max content width 1400px. Tables show all columns. Side panels (slide-overs) open at 480px wide from right.
Tablet (768–1199px)
Sidebar collapses to 64px (icons only). Hover to see labels. Content fills remaining width. Tables scroll horizontally. Side panels full-screen on smaller tablets.
Mobile (<768px) — limited
Admin panel is NOT mobile-first. Mobile shows a warning: "For best experience, use a desktop or tablet." Basic read-only views still work. No table editing on mobile.
Sidebar navigation (always visible on desktop)
Top: App logo + "Admin Panel" label
Nav links (with icons): Dashboard · Categories · Products · Vendors · Orders · Payments · Payouts · Users · Coupons · Search Insights
Active state: brand-red left border + brand-light background on active nav item
Bottom: Admin name + avatar initials circle + "Sign out" link
Sidebar is fixed — does not scroll with content
Global header bar (top of every page)
Page title (e.g. "Dashboard") in 20px semibold — changes per screen
Breadcrumb below title where applicable: "Home / Vendors / Pending"
Right side: notifications bell (admin-specific alerts) + admin avatar
Notifications: new vendor registration, failed payments needing review, payout due alerts
Design system — admin-specific
Colors (same brand, slightly different usage)
Brand red #E8563C for primary actions only (Approve, Save). Tables use #F9FAFB row hover. Status badges: green=active/success, amber=pending/warning, red=suspended/failed, gray=inactive. Danger zone actions: red outlined buttons.
Typography + Components
Plus Jakarta Sans. Tables: 13px rows, 11px column headers. Cards: 15px body. Shadcn/ui Table, Dialog, Sheet, Select, Command components. Recharts for all charts. TanStack Table for sortable/filterable tables.
Slide-over panel pattern (used everywhere)
All add/edit/detail actions open a slide-over panel from the right — never a new page
Slide-over: 480px wide on desktop, full-screen on mobile/small tablet
Backdrop: semi-transparent overlay behind panel, click to close
Panel has: header (title + close ✕) · scrollable body · sticky footer (Cancel + action button)
Form in slide-over has unsaved changes? Closing shows "Discard changes?" confirmation
Tech stack
React 18 + Vite + TypeScript
Tailwind CSS + Shadcn/ui (Table, Dialog, Sheet, Badge, Select, Command, DatePicker)
TanStack Query v5 — server state, auto-refetch on window focus
TanStack Table v8 — sortable, filterable, paginated tables with column visibility toggle
Recharts — line charts, bar charts, funnel charts on dashboard
React Hook Form + Zod — all forms
React Router v6 — nested routes, protected by admin auth guard
Zustand — admin auth token (in memory only)
